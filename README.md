# Gmail Clone Project

This project is a static web page that replicates the user interface of Gmail. Built using HTML and Tailwind CSS, it demonstrates the core design and layout of the Gmail email client.

## Features

- Home page
- Inbox view
- Email view

## Technologies Used

- HTML
- Tailwind CSS
